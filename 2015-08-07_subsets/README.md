Given a set of distinct integers, return all possible subsets.

- The elements in a subset must be in an ascending order.
- There should be not duplicate subsets.
- The subsets can be in any order.
 
```
Input: [1, 2, 3, 4]
Output:
[]
[1]
[2]
[3]
[4]
[3, 4]
[2, 3]
[1, 2]
[2, 4]
[1, 3]
[1, 4]
[1, 2, 3]
[1, 3, 4]
[1, 2, 4]
[2, 3, 4]
[1, 2, 3, 4]
 
Input: [0, 4, 1]
Output:
[]
[0]
[1]
[4]
[0, 1]
[0, 4]
[1, 4]
[0, 1, 4]
```
