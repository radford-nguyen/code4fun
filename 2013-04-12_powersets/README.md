If S is the set {x, y, z}, then the subsets of S are:

```
{} 

{x}

{y}

{z}

{x, y}

{x, z}

{y, z}

{x, y, z}
```

and hence the power set of S is

    {{}, {x}, {y}, {z}, {x, y}, {x, z}, {y, z}, {x, y, z}}

More information: http://en.wikipedia.org/wiki/Powerset
